# Card color scheme: green/orange/red only for status values, icons/titles match background

Card icons, titles, and decorative elements should use a neutral color (matching background/secondary). Only the metric value/number should use green (success), orange (warning), or red (alert) based on status. Currently too many elements are green which looks overwhelming.
