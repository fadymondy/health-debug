# Fix primary button color — green on white is harsh on eyes

AppTheme.primary (neon green #00FF41 dark / #034D22 light) is used as tint on .glassProminent and .glass buttons. The bright green with white label text is hard on the eyes. Need to pick a more comfortable alternative — e.g., a muted teal/blue, or use the secondary color (#00D4FF) as the action color instead.
