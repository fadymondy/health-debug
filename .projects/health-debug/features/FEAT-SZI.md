# Smart Meal Planner — Full Plate UI

Replace the quick-log grid in NutritionView with a full meal planner. UI: a visual plate divided into sections (protein half, carb quarter, fat quarter — like MyPlate). User taps a section to pick food from the safe list or type custom. AI classifies custom entries in real time (GERD risk, gout risk, etc.). After building the plate, tap 'Log Meal' — AI generates a smart comment ('Good choice — low GERD risk. Watch portion size on the rice.'). Show today's meals as visual meal cards (breakfast/lunch/dinner/snack). Each meal card has an AI 'Analyze' button.
