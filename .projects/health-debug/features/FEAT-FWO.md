# GERD & Sinus System Shutdown Timer

4-hour pre-sleep fasting countdown based on user's target sleep time. 'System Shutdown' mode UI state. Allowed during shutdown: water, chamomile tea, anise tea. If food is logged during shutdown window: throw 'GERD/Sinus Flare-up Risk' exception warning. Tech tone: 'System Shutdown initiated — kernel cooling in progress.'
