# Smart Card Detail Pages with AI Actions

Every dashboard card (Hydration, Caffeine, Nutrition, Shutdown, Stand, Sleep, Steps, Heart Rate, Zepp) opens a full detail page. Each page has: logging UI, trend chart, AI insight section, and smart action buttons (e.g. 'Why is my heart rate high?' 'What should I eat now?' 'Should I drink water?'). Actions call AIInsightEngine with real-time context — not static CRUD.
