# Focus & Red Bull Deprecation Tracker

Rule 1: Block caffeine logging for first 90-120 min after waking (adenosine clearance). Rule 2: Track transition from sugar-based energy drinks (Red Bull) to clean caffeine (Cold Brew, Matcha, Green Tea). Rule 3: 'Fatty Liver Alert' on sugar-based caffeine logging. CaffeineLog model with type enum and isSugarBased flag.
